The `config` folder contains the core configuration components for the **LLMOps Multi-AI Agent** project.
Its purpose is to centralise environment loading, API keys, and any global settings the application requires.

## Files

### **settings.py**

Provides the main configuration class for the project.
It loads environment variables using `.env`, exposes API keys, and defines allowed model names for the agent.
